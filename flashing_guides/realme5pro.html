UBL & TWRP Guide

  1. Unlock Your Bootloader, you can visit this link for guide Unlocking Bootloader or watch youtube videos
  2. After Unlock Bootloader, Download all the necessary file requires in your PC
  3. Put twrp.img and vbmeta.img in to adb platform tools folder
  4. Open command window here from platform tools folder
  5. fastboot flash vbmeta vbmeta.img
  6. fastboot flash recovery (twrp name).img

Flashing Rom (custom rom / stock rom)

  1. Reboot recovery by pressing power + vol down, use volume button choose reboot recovery. Press power to boot in to recovery mode
  2. If you see a random folders name you need to format data
  3. Copy paste rom files to internal storage
  4. Wipe all partition except internal / factory reset
  5. Flash the Rom / Stock Rom what you want to flash
  6. (For Stock Rom flashing ) if you see message like "Mount :failed to mount /dev/block/bootdevice/by-name/userdata at /data:Device or resource busy" . JUST IGNORE , IT IS NOT FAIL !!!
  7. before reboot kindly check the notes below 👇 , make sure you have done everything , Then Reboot to system
Notes
If you want twrp stay after flashing stock rom ( Realme UI ) flash vbmeta.img and mount /vendor then open file manager go to /vendor delete recovery.from.boot.p
If you want to flash a custom ROM then you need to flash the firmware, make sure you read the ROM post notes before flashing, does the ROM need COS or Realme UI firmware (if u didn't flashed the firmware before going to custom rom u might get error while flashing) if no error exist during flash ignore this
If you are Stuck in Recovery mode / always back to twrp , you need to format data or go to /data delete all folder and file inside it except folder media
Or Stuck in fastboot mode , you need to flash dtbo.img via twrp ( this case is rarely happen )
